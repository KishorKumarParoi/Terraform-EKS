# Roadmap

## Phase 1: Foundation

- Build cloud-specific entrypoints for AWS and Azure.
- Define shared module patterns.
- Split environments into dev, staging, and prod.

## Phase 2: Platform Services

- Add networking, compute, and security modules.
- Add Kubernetes platform layers.
- Add secret management and IAM / RBAC patterns.

## Phase 3: Delivery Automation

- Add CI/CD workflow files.
- Add Terraform validation, linting, and security scans.
- Add promotion rules and deployment gates.

## Phase 4: Observability and Operations

- Add monitoring dashboards and alerting.
- Add logs and operational runbooks.
- Add backup and restore patterns.

## Phase 5: Resume Polish

- Add diagrams.
- Add demo commands.
- Add architecture notes and troubleshooting steps.
- Add a short case study summary for your CV.
